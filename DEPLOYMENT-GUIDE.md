# 📋 GUÍA DE DEPLOYMENT - Diego Roman Portfolio

Tu portfolio está 100% listo para publicar. Aquí están los pasos:

## 🚀 OPCIÓN 1: Desplegar en Vercel (RECOMENDADO)

### Paso 1: Crear cuenta en GitHub
1. Ve a https://github.com/signup
2. Crea tu cuenta (si no la tienes)
3. Verifica tu email

### Paso 2: Crear repositorio en GitHub
1. Ve a https://github.com/new
2. Nombre del repo: `diego-portfolio`
3. Descripción: "Diego Roman - Growth Marketer & Brand Strategist Portfolio"
4. Selecciona "Public"
5. Click en "Create repository"

### Paso 3: Subir código a GitHub
Copia estos comandos en tu terminal:

```bash
cd diego-portfolio-repo  # (O la carpeta donde esté tu portfolio)
git remote add origin https://github.com/TU_USUARIO/diego-portfolio.git
git branch -M main
git push -u origin main
```

### Paso 4: Desplegar en Vercel
1. Ve a https://vercel.com/signup
2. Haz click en "Continue with GitHub"
3. Autoriza Vercel a acceder a GitHub
4. Click en "Import Project"
5. Selecciona el repo `diego-portfolio`
6. Vercel auto-detectará la configuración
7. Click en "Deploy"

**¡LISTO!** Tu portfolio estará en vivo en 1 minuto en una URL como:
`https://diego-portfolio.vercel.app`

---

## 🔗 ENLACE CORTO
Puedes compartir tu portfolio en formato corto:
- `diego-portfolio.vercel.app`
- O tu dominio personal: compra uno en Vercel

## 📝 PERSONALIZAR URL EN VERCEL
1. Ve a tu proyecto en Vercel
2. Proyecto Settings → Domains
3. Agregue dominio personalizado o usa el gratuito
